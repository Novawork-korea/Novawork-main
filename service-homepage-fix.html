이런 의뢰에 적합합니다

  • 홈페이지 일부가 깨지거나 모바일에서 이상하게 보이는 경우
  • 텍스트·이미지·연락처·버튼 링크를 수정해야 하는 경우
  • 섹션, 버튼, 팝업, 배너 등 간단한 기능을 추가하고 싶은 경우
  • 외주 제작 사이트의 간단한 유지보수가 필요한 경우
02

작업 가능 범위

  • HTML/CSS 레이아웃 및 반응형 화면 수정
  • 이미지, 문구, 링크, 버튼, 헤더·푸터 메뉴 수정
  • 간단한 섹션 추가
  • 팝업, 토글, 탭, 아코디언 등 간단한 인터랙션
  • JavaScript 오류 확인과 배포 전후 검수

    수정 URL·오류 화면·변경 내용 확인

  3. 03

    원인과 수정 범위 진단

  4. 04

  • 소스코드 또는 GitHub/FTP/호스팅 접속 정보
