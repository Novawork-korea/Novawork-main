웹시스템 개발

React·Firebase 기반 DB·관리자 웹시스템 개발

사용자가 입력한 데이터를 저장하고 관리자가 처리하는 운영형 웹시스템을 만듭니다.

React 프론트엔드와 Firebase 백엔드를 조합해 신청폼, 예약 관리, 관리자 페이지, 로그인, 파일 업로드, 검색·필터·상태 관리를 구현합니다.

SERVICE FLOW

고객 입력 데이터가 저장되고 관리되는 운영형 웹사이트입니다.

폼으로 받은 신청·예약·상담 데이터를 Firebase에 저장하고, 관리자가 목록·상세·상태·메모를 확인할 수 있는 구조로 구현합니다.

  • 02관리자 화면 구성
  • 03권한·상태 관리
01

이런 의뢰에 적합합니다

  • 신청서·예약·상담 접수·견적 요청을 자체 시스템으로 받고 싶은 경우
  • 고객 정보를 관리자 화면에서 조회·수정·처리해야 하는 경우
  • 내부 직원용 업무 관리자 페이지가 필요한 경우
  • 초기 스타트업·MVP·검증용 웹서비스가 필요한 경우
02

작업 가능 범위

  • React 기반 사용자 화면과 관리자 화면 개발
  • Firebase Firestore 데이터 저장 구조 설계
  • Firebase Authentication 로그인·권한 처리
  • Firebase Storage 이미지·파일 업로드
  • 검색, 필터, 정렬, 상태값 변경, 메모, 상세 보기, 수정/삭제, CSV/엑셀 다운로드
03

진행 방식

  1. 01

    요구사항·사용자 유형·관리자 역할 확인

  2. 02

    필수 기능과 선택 기능 분리

  3. 03

    데이터 구조와 화면 구조 설계

  4. 04

    React/Firebase 개발 및 테스트 링크 공유

  5. 05

    기능 오류·화면 조정 후 배포와 사용법 안내

04

의뢰 전 준비하면 좋은 자료

  • 시스템 목적과 사용자 유형
  • 저장해야 할 데이터 항목과 필요한 화면 목록
  • 관리자 처리 운영 흐름
  • 로그인·파일 업로드·알림·결제·API 연동 필요 여부
  • Firebase/Google 계정 및 외부 서비스 계정 정보
05

제외 또는 별도 협의 범위

  • 대규모 엔터프라이즈 시스템, ERP/그룹웨어 전체 구축
  • 네이티브 앱 개발, 고난도 실시간 채팅·대규모 트래픽 구조
  • 복잡한 PG·정산·회계 시스템, 전문 보안 컨설팅
