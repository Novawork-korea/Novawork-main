ESTIMATE LOGIC

같은 홈페이지라도 가격이 다른 이유

가격은 단순 페이지 수만으로 정하지 않습니다. 화면 수, 기능 수, 자료 준비 상태, 반응형 난이도, 추적/운영 범위에 따라 작업량이 달라집니다.

화면 수

메인 1페이지인지, 서브 페이지가 포함되는지에 따라 작업량이 달라집니다.

기능 수

문의폼, 예약폼, 관리자 화면, 로그인, 파일 업로드, 외부 API 연동 여부를 확인합니다.

자료 준비 상태

문구, 이미지, 로고, 참고 사이트, 계정 권한이 준비되어 있으면 일정이 짧아집니다.

반응형 난이도

PC와 모바일에서 같은 정보가 자연스럽게 보여야 하므로 화면 구조에 따라 작업량이 달라집니다.

추적 / 운영 범위

PROCESS

작업은 세 단계로 단순하게 진행합니다.

상담 단계에서 범위를 과하게 키우지 않고, 현재 필요한 결과물부터 먼저 정리합니다.

STEP 01

문제와 목적 분리

대표 링크, 관리자 시스템, 기존 사이트 수정, 측정, AI 챗봇, 데이터 수집 중 어디에 해당하는지 먼저 나눕니다.

STEP 02

구조와 준비 자료 확정

화면 수, 기능 수, 데이터 항목, 계정 권한, 참고 자료, 검수 기준을 정리한 뒤 작업 범위를 확정합니다.

STEP 03

구현·검수·전달

테스트 링크 또는 결과 파일을 공유하고, 합의된 범위 안에서 수정 후 최종 파일과 안내를 전달합니다.
